OS & Network

This page contains my OS Kerenel and Network experinces

  • As development Engineer for LOGBAX (acquired by Olivetti) I have participate in the development and was the sole maintainer of an multi-user operating system for a mini-computer called LX-6000. OS was written in a very low level programming language (micro-programming).

  • While working for Capgemini, I was involved with the development of a real-time OS and interfacing between OS team and telephony application team using that OS, helping them with system calls and architecture related to the OS use. I was working in our customer site "Jeumont-Schneider".

  • While working for Copernique; one of Capgemini client; I was involved in development (mainly performance improvement) of another real-time and distributed OS based on UNIX kernel.

  • While working for LUTRANA, I developed a small RTOS for an embedded system using directly hardware interrupt of the microprocessor.

  • Implemented multiple network protocols at layer 2,3,4 and application layer. Please see the list of projects in Software tab.

  • Experience at protocol level: TCP/IP, UDP, HTTP, HTTPS, ICAP, SOAP
